# Portfolio

My [portfolio website](https://portfolio-rajan-phadnis.web.app/), designed and built by me in Flutter.


## Site Info

Pushes or pull requests to the "dev" branch deploy to a private Firebase preview channel site.

Pushes or pull requests to the "master" branch deploy to [portfolio-rajan-phadnis.web.app](https://portfolio-rajan-phadnis.web.app/) and [portfolio-rajan-phadnis.firebaseapp.com](https://portfolio-rajan-phadnis.firebaseapp.com/). Both are identical redirects to the same website.
